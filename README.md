# Менеджер досок / Канбан менеджер / Trello clone

_Менеджер досок, с возможностью изменять имена существующих boards и cards, и сохранением данных в localStorage_

## Список технологий

- React
- Redux
- React Router
- redux-localstorage-simple
- Material UI
